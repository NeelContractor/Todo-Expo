This is a simple to-do application built with React Native Expo. It includes functionality to create tasks, mark them as complete, and delete them.

## Steps to Start

1. Clone the repo locally
```bash
git clone <repo_url>
```
2. install dependencies
```bash
npm install
```
3. run the project
```bash
npx expo start
```
4. open expo go app on your mobile and scan the qrcode to run the project locallly.
